Chef Muscle

Chef Muscle is een applicatie die ik met mijn groepje heb ontworpen voor een bodybuilder. Je kan de maaltijden die je hebt gegeten en je sportactiviteiten invoeren in de app. Er waren nog meer schermen, maar deze waren het belangrijkst. Deze applicatie is niet gebouwd met code.

Plog De Dag

Plog De Dag was een opdracht voor de gemeente Amsterdam. De opdracht was maak een pagina (Plog De Dag) voor de website van Amsterdam. Het ziet er niet heel bijzonder uit, omdat het moest perse bij de huisstijl van de gemeente Amsterdam passen. Dit is ook alleen visueel, geen code.

Brawl Bibliotheek

Hieronder zie voorbeelden van schermen die ik moest ontwerpen voor een biblotheek op een middelbareschool. Het doel was op de onderbouw meer aan het lezen te krijgen. Ik had als inspiratie het spel Brawl Stars. Ik had ook andere versies, maar die waren te serieus voor de doelgroep. Dit is alleen visueel, geen code.

Techno Holiday

Ik heb de applicatie Techno Holiday ontworpen. Je mocht zelf kiezen wat je ging doen. Ik heb hiervoor gekozen, omdat ik graag naar festivals ga. De bedoeling van de applicatie is dat je kan zien waar feesten zijn. Je kan ook een plaats invullen en dan zie je alle feesten die daar in de buurt zijn. Hieronder zie je de belangrijkste schermen. Alleen visueel, geen code.

De Slimme Koelkast

Ik en mijn groepje moesten iets bedenken om frustraties rondom de koelkast te beperken in een studentenhuis. Het concept is dat je doormiddel van een vingerprint kan bijhouden wie wat pakt uit de koelkast. Alles op deze poster heb ik gemaakt. In de praktijk slaat dit concept natuurlijk nergens op.

Sven's Restaurant

Smelly Roads

Smelly Roads is een interactieve poster over de geuren die je ruikt onderweg naar school. Hoe hoger deste viezer de geur was. Als je over de geuren heen hovert zag je de geur. Dit is in figma gemaakt, geen code.
